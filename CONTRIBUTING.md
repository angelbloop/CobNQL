# CONTRIBUTING

Спасибо за интерес к CobNQL! 🙏

## Как контрибьютить

### Сообщение о багах
- Откройте Issue с заголовком `[BUG]`
- Включите версию Python и шаги воспроизведения

### Фич-реквесты
- Откройте Issue с заголовком `[FEATURE]`
- Объясните use case

### Код

1. Fork репозиторий
2. Создайте бранч (`git checkout -b feature/my-feature`)
3. Коммитьте (`git commit -m 'feat: description'`)
4. Пушьте (`git push origin feature/my-feature`)
5. Откройте PR

## Требования к коду

- PEP 8 стиль
- Type hints
- Docstrings для функций
- Протестировано локально
- Нет чувствительных данных

## Лицензия

Контрибьютируя, вы согласны лицензировать код под MIT License.
